# asunaroblog
This is my personal blog ([asunaroblog](https://asunaroblog.net/)) repository. It's built with Next.js and Newt (Jamstack).

## UI design
- [MUI](https://mui.com/)
  - To build common components like a sidebar.
- [Tailwind CSS](https://tailwindcss.com/)
  - To style raw HTML codes that are parsed with Cheerio.
 
## CMS (Contents Management System)
- [Newt](https://www.newt.so/)
    - It's a very convenient and highliy avalable CMS that is made in Japan.

## Hosting service
 - [Cloudflare Pages](https://pages.cloudflare.com/)
    - It has a high performance CDN.
    - We can use many functions and resources for free.
